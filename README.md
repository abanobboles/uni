Dieses Repository sammelt die Unterlagen fuer mein Theologiestudium.

Die Materialien aus den einzelnen Faecherordnern sollen hier so organisiert sein, dass gemeinsames Lernen, Zusammenfassen, Bearbeiten von Aufgaben und Erstellen neuer Inhalte moeglich ist.

- `Notes`
