# Theologiestudium Repository

Dieses Repository sammelt die Unterlagen fuer mein Theologiestudium.

## Zweck

Die Materialien aus den einzelnen Faecherordnern sollen hier so organisiert sein, dass gemeinsames Lernen, Zusammenfassen, Bearbeiten von Aufgaben und Erstellen neuer Inhalte moeglich ist.

## Faecheruebersicht

1. `Theologie Altes Testament IIa. Geboren werden, lieben, leiden, sterben. Lebensphasen im Alten Testament`

## Struktur pro Fach

Jeder Fachordner soll diese Struktur haben:

- `Moodle`
- `Notes`
- `DB.md`

## Wichtige Hinweise aus den Fach-DBs

### `Theologie Altes Testament IIa. Geboren werden, lieben, leiden, sterben. Lebensphasen im Alten Testament`

- `DB.md` ist aktuell noch nicht angelegt.
- Sobald eine `DB.md` existiert, sollen hier die wichtigsten Punkte knapp gespiegelt werden.
- Besonders relevant sind:
  - offene Hausaufgaben
  - Abgaben und Fristen
  - Pruefungstermine
  - wichtige organisatorische Hinweise
  - besondere Lernhinweise oder offene Punkte

## Pflege der README

- Wenn neue Faecher hinzukommen, sollen sie hier in die Faecheruebersicht aufgenommen werden.
- Wenn sich wichtige Informationen in einer fachbezogenen `DB.md` aendern, soll die README im Ueberblick mitaktualisiert werden.
- Die README ist eine kompakte Startseite und ersetzt nicht die fachspezifischen `DB.md`-Dateien.
