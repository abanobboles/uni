# 4. Das Konzil von Nizäa

### Das ökumenische Konzil von Nizäa, 325
 
Das Konzil von Nizäa ist das erste ökumenische Konzil der Kirchengeschichte. Auslöser ist ein Streit um die Frage, wer Jesus Christus eigentlich ist: wahrer Gott oder höchstes Geschöpf?
Mit dem Bekenntnis zur Wesensgleichheit von Vater und Sohn wird eine Grundlage gelegt, die das christliche Glaubensverständnis bis heute prägt. Nizäa zeigt exemplarisch, wie Glaubenslehre im Spannungsfeld von Bibelauslegung, Philosophie und kirchlicher Einheit entsteht.
 
**Leitfragen zum Podcast**
 
- Warum wird aus einer theologischen Debatte ein Konzil des ganzen Reiches?
- Was steht auf dem Spiel, wenn über die Gottheit Christi gestritten wird?
- Wie entstehen Glaubensbekenntnisse – und warum sind sie verbindlich?
